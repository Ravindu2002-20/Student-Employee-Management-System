# Student-Employee-Management-System
This project is a web-based system to manage student and employee information with secure login and role-based access.
When a user visits the website, they first see a login page. After logging in, they can view lists of students and employees in tables. Normal users can only view details, while admins have special permissions to add, edit, and delete records.

The system is built using:
Next.js for the frontend (user interface)
NestJS for the backend (API)
PostgreSQL as the database
Prisma to manage database operations
JWT authentication to keep user access secure

The main goal of this project is to create a secure and organized platform where data can be managed efficiently, while ensuring that only authorized users can make changes.
